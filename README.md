# AI Debug Coach
An AI-powered debugging assistant that analyzes C# code errors using the Gemini API.

---

## Key Features
- Real-time AI analysis of code and error messages.
- Persistent debugging history using SQLite & Entity Framework Core.
- Clean architecture with separation of concerns.

---

## Getting Started
1. Clone the repo.
2. Run `dotnet user-secrets init`.
3. Run `dotnet user-secrets set "GeminiApiKey" "YOUR_API_KEY"`.
4. Run `dotnet run`.
